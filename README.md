# Modelling Project

## Overview
This project focuses on building and evaluating machine learning models for data analysis and prediction. It includes experiments, notebooks, and scripts to explore different modelling techniques.

## Features
- Data preprocessing and cleaning
- Exploratory data analysis (EDA)
- Implementation of multiple machine learning algorithms
- Model evaluation and comparison
- Visualization of results
