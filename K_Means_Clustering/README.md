# K-Means Clustering Project

## 📌 Overview
This repository contains a Jupyter Notebook that demonstrates the use of **K-Means Clustering**, an unsupervised machine learning algorithm. The project explores how data points can be grouped into clusters based on similarity, and visualizes the results for better understanding.

---

## 📂 Project Structure
- `K_Means_Clustering.ipynb` → Notebook implementing K-Means clustering
- `requirements.txt` → Python dependencies
- `.gitignore` → Ignored files and folders
- `README.md` → Project documentation
