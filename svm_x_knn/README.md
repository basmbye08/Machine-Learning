# Breast Cancer Classification using KNN and SVM

A machine learning project that compares the performance of K-Nearest Neighbors (KNN) and Support Vector Machine (SVM) classifiers for breast cancer prediction.

## Features
- Data preprocessing
- Feature scaling using StandardScaler
- KNN Classification
- SVM Classification
- Model Evaluation
- Performance Comparison
- Data Visualization

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the notebook:

```bash
jupyter notebook KNN_x_SVM.ipynb
```

## Project Workflow
1. Load the dataset.
2. Split the data into training and testing sets.
3. Scale the features.
4. Train the KNN model.
5. Train the SVM model.
6. Make predictions.
7. Evaluate and compare both models.
